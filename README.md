**Arrow Function Demo (JavaScript)**

This project demonstrates the usage of **JavaScript Arrow Functions** with different parameter types and use cases.  
The outputs are displayed using **console.log()** for learning and practice purposes.

** Purpose of the Project**
- To understand arrow function syntax
- To practice writing clean ES6 JavaScript
- To observe output in the browser console
- To learn functional concepts like callbacks (`map`)
